# Personal Blog

## Description
This project is a personal blog that showcases my thoughts, experiences, and ideas. It is designed to be visually appealing and easy to navigate.

## Technologies
- HTML
- CSS
- JavaScript
- Node.js
- Express
- MongoDB

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/JHY-CPU/personal-blog.git
   ```
2. Navigate to the project directory:
   ```bash
   cd personal-blog
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application:
```bash
node app.js
```
Then navigate to `http://localhost:3000` in your browser to access the blog.

## Features
- User Authentication
- Commenting System
- Responsive Design
- Dark Mode and Light Mode
- Tags and Categories for Posts

## Project Structure
```
personal-blog/
│
├── models/        # Database models
├── routes/        # Express routes
├── views/         # Template files
├── public/        # Static files (CSS, JS, images)
└── app.js         # Main entry point
```
